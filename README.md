# Scientific Calculator

A fully functional Scientific Calculator built with JavaScript, HTML, and CSS.
No frameworks. No libraries. Just clean, handwritten code.

## Features

- Basic operations — addition, subtraction, multiplication, division
- Scientific functions — sin, cos, tan, square root
- Clean, responsive UI
- Error handling for invalid inputs

## Tech Stack

- HTML5
- CSS
- JavaScript

## Why I Built This

I'm a Python Full Stack Developer who doesn't believe in stack boundaries.
This project was a deliberate push into frontend territory —
to prove I can pick up new tools, ship real output, and write clean code
outside my primary stack.

## Status

✅ v1 — Functional and live  
🔄 v2 — In progress. Smoother UI, better logic, more features coming.

## Run Locally

No setup needed. Just clone and open.

git clone https://github.com/nandagopal-pn/scientific-calculator.git

Then open index.html in your browser. Done.
